# Level_01_Basics

A collection of fundamental C++ programs covering essential programming concepts and operations.

## Contents

- **01_even_odd.cpp** - Determines whether a number is even or odd
- **02_swaping.cpp** - Swaps two variables using different methods
- **03_largest_num.cpp** - Finds the largest number among given inputs
- **04_calculate_interest.cpp** - Calculates simple or compound interest
- **05_temperature_converter.cpp** - Converts between Celsius and Fahrenheit
- **06_reverse_number.cpp** - Reverses the digits of a number
- **07_multiplication_table.cpp** - Generates multiplication tables
- **08_factorial.cpp** - Calculates the factorial of a number
- **09_prime_number.cpp** - Checks if a number is prime
- **10_sum_of_numbers.cpp** - Computes the sum of a series of numbers
- **11_stars_pattern.cpp** - Prints various star patterns

## Purpose

This repository contains introductory C++ exercises designed to build foundational programming skills including conditionals, loops, functions, and basic algorithms.
